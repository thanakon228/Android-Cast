# 📡 Android Cast

> แคสหน้าจอมือถือ **Android ขึ้นหน้าต่างบน PC แบบไร้สาย** — หน้าตาและการเชื่อมต่อสไตล์ **TikTok Live**

โปรแกรมนี้เขียนด้วย **Python + PyQt6** ครอบ [scrcpy](https://github.com/Genymobile/scrcpy) + adb ไว้
จุดเด่นคือ **ดาวน์โหลด scrcpy ให้อัตโนมัติ** ตอนเปิดครั้งแรก คุณไม่ต้องติดตั้ง scrcpy หรือ adb เอง

<p align="center">
  <em>เชื่อมต่อครั้งเดียว → ครั้งต่อไปกดปุ่มเดียวแคสได้เลย</em>
</p>

---

## ✨ ฟีเจอร์

- 🔌 **ดาวน์โหลด scrcpy + adb อัตโนมัติ** (~15MB) — ไม่ต้องตั้งค่าอะไรเอง
- 📶 **เชื่อมต่อไร้สาย** ผ่าน Wireless debugging (Android 11+) ด้วยรหัสจับคู่ 6 หลัก
- 🔍 **ค้นหาอุปกรณ์อัตโนมัติ (mDNS)** — เติม IP ให้เอง เหลือกรอกแค่รหัส 6 หลัก
- 🔁 **ต่อใหม่อัตโนมัติ (auto-reconnect)** — หลุดแล้วต่อกลับเองพร้อม backoff โดยที่ถ้าคุณปิดหน้าต่างมิเรอร์เองจะหยุดให้ ไม่เด้งกลับมา
- ⚡ **เชื่อมต่อด่วน** — จำอุปกรณ์ล่าสุด ครั้งต่อไปกดปุ่มเดียวจบ
- 🔌 **โหมด USB** สำรอง — เสียบครั้งเดียวแล้วสลับมาไร้สายอัตโนมัติ (รองรับ Android เก่ากว่า 11)
- 🖥️ **Console ในตัวโปรแกรม** — เห็น log ของ adb/scrcpy/การเชื่อมต่อแบบเรียลไทม์
- 📊 **สถานะเรียลไทม์บนแถบ live** — 📱 ชื่อรุ่นมือถือ · 🔋 แบต · 🎞️ FPS · 📶 ping · ⚙️ bitrate
- 🎛️ **ปุ่มควบคุมขณะแคส** — 🔄 หมุนจอ · 🔇 ปิด/เปิดเสียง · 📸 แคปจอ · 🔌 ตัดการเชื่อมต่อ
- 🎚️ **แผงตั้งค่าคุณภาพ** — พรีเซ็ต ลื่นสุด/สมดุล/คมสุด + ปรับ bitrate/FPS/ความละเอียดเอง
- 🔴 **อัดวิดีโอหน้าจอ** เป็น .mp4 + 📸 **แคปหน้าจอ** เป็น .png
- 🔊 **ส่งเสียงมือถือมาที่ PC** · 🌙 **ปิดจอมือถือขณะแคส** · ☕ **กันจอหลับ** · 📌 **Always-on-top**
- 🖼️ **ฝังหน้าจอในแอป** (ทดลอง) — รวมเป็นหน้าต่างเดียว
- 🎥 **โหมดสตรีม OBS** — หน้าต่างชื่อคงที่ ให้ OBS Window Capture เลือกเจอและล็อกได้แม้ต่อใหม่
- 📦 **สร้างไฟล์ .exe** แจกได้โดยไม่ต้องลง Python
- 🎨 UI ธีมดำ-ชมพู สไตล์ TikTok Live

| รายการ | รายละเอียด |
|--------|-----------|
| ระบบปฏิบัติการ | Windows 10/11 (64-bit) |

| เครือข่าย | มือถือกับ PC ต้องอยู่ **WiFi วงเดียวกัน** |

> 💡 ตอนติดตั้ง Python อย่าลืมติ๊ก **"Add Python to PATH"**

## 🚀 การติดตั้ง

### 1. โคลนโปรเจกต์
```bash
git clone https://github.com/thanakon228/Android-Cast.git
cd Android-Cast
```

### 2. สร้าง virtual environment + ติดตั้งไลบรารี
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### 3. เปิดโปรแกรม
```bash
python app.py
```
หรือดับเบิลคลิกที่ไฟล์ **`run.bat`** ได้เลย

> ⏳ เปิดครั้งแรกโปรแกรมจะดาวน์โหลด scrcpy อัตโนมัติ รอจนแถบสถานะขึ้นสีเขียว **"พร้อมใช้งาน"**

## 📱 วิธีใช้งาน

### โหมด WiFi (ไร้สาย) — Android 11 ขึ้นไป ✅ แนะนำ
1. มือถือ → **Settings → About phone** → แตะ **Build number** 7 ครั้ง (เพื่อเปิด Developer options)
2. **Settings → Developer options → Wireless debugging** → เปิด
3. แตะ **"Pair device with pairing code"** → จะได้ `IP:PORT` และ **รหัส 6 หลัก**
4. (ทางลัด) กด **"🔍 ค้นหาอุปกรณ์ในวง WiFi"** แอปจะเติม `IP:PORT` ให้อัตโนมัติ เหลือกรอกแค่รหัส
5. กด **"จับคู่ & เชื่อมต่อ & แคส"** → หน้าจอมือถือเด้งขึ้นมาเป็นหน้าต่างบน PC 🎉

ครั้งต่อ ๆ ไป โปรแกรมจะจำไว้ → กดปุ่ม **"เชื่อมต่อด่วน"** ปุ่มเดียวจบ

### โหมด USB — สำหรับ Android เก่ากว่า 11
1. เปิด **Developer options → USB debugging**
2. เสียบสาย USB → กด **"Allow"** บนมือถือ
3. กด **"ตรวจหาอุปกรณ์ USB"** → **"เปิดไร้สาย & แคสเลย"**
4. ถอดสายได้เลย ใช้งานไร้สายต่อ

## ⚠️ ข้อควรรู้

- scrcpy/adb **จำเป็นต้องเปิด Developer options ครั้งแรกเสมอ** — เลี่ยงไม่ได้
  หากต้องการแคสโดยไม่แตะ Developer options เลย ต้องเขียนแอปฝั่ง Android เองที่ใช้ MediaProjection API (เป็นคนละแนวทาง)
- มือถือกับ PC ต้องอยู่ **WiFi วงเดียวกัน**
- การเชื่อมไร้สายล้วน (pairing code) ต้องใช้ **Android 11 ขึ้นไป**

**ตัวเลือกเพิ่มเติม**

| ตัวเลือก | ผล |
|----------|-----|
| 🔴 อัดวิดีโออัตโนมัติ | บันทึกเป็น `.mp4` ไว้ที่โฟลเดอร์ `recordings/` ขณะแคส |
| 🔊 ส่งเสียงมาที่ PC | ฟังเสียงมือถือบน PC (Android 11+) |
| 🌙 ปิดหน้าจอมือถือ | ดับจอมือถือขณะแคส (ประหยัดแบต/กันแอบดู) |
| ☕ คาหน้าจอไม่ให้หลับ | ลดอาการหลุดจากการที่มือถือเข้าสลีป |
| 📌 Always-on-top | ปักหน้าต่างมิเรอร์ไว้บนสุด |
| 🖼️ ฝังหน้าจอในแอป | (ทดลอง) รวมจอมือถือเข้ามาในหน้าต่างแอป |
| 🎥 โหมดสตรีม OBS | หน้าต่างแยกที่มี **ชื่อคงที่ `AndroidCast - Live`** ให้ OBS เลือกเจอและล็อกไว้ได้แม้ต่อใหม่ |

> 📸 ปุ่ม **"แคปจอ"** อยู่บนแถบ live ขณะแคส — บันทึกภาพไว้ที่โฟลเดอร์ `screenshots/`

### 🎥 ใช้กับ OBS / โปรแกรมสตรีม
- **แนะนำ:** เปิด **"🎥 โหมดสตรีม OBS"** ในแท็บตั้งค่า → ใน OBS เพิ่ม **Window Capture** แล้วเลือกหน้าต่าง **`AndroidCast - Live`**
  ชื่อหน้าต่างคงที่ ทำให้ OBS ล็อกไว้ได้แม้แคสหลุดแล้วต่อใหม่
- ถ้าใช้ **"🖼️ ฝังหน้าจอในแอป"** OBS จะมองไม่เห็นจอมือถือ (เพราะเป็นหน้าต่างลูก) — กรณีนี้ให้จับทั้งหน้าต่างแอป **ScreenCast Studio** ด้วยวิธีจับภาพ **"Windows 10 (1903 and up)"** แทน
- หมายเหตุ: โหมดสตรีมกับฝังในแอป **เปิดพร้อมกันไม่ได้** (เลือกอย่างใดอย่างหนึ่ง)

## 📦 สร้างไฟล์ .exe (แจกให้คนอื่นใช้โดยไม่ต้องลง Python)

ดับเบิลคลิก **`build_exe.bat`** (หรือสั่ง `python -m PyInstaller --onefile --windowed --name AndroidCast app.py`)
ได้ไฟล์ที่ **`dist\AndroidCast.exe`** — เปิดได้เลย (scrcpy จะถูกดาวน์โหลดอัตโนมัติตอนรันครั้งแรก)

### 🤖 Build อัตโนมัติด้วย GitHub Actions
มี workflow ที่ `.github/workflows/build-exe.yml` ทำ build ให้บนคลาวด์:
- **อยากออกเวอร์ชันใหม่** → สร้าง tag ขึ้นต้นด้วย `v` แล้ว push:
  ```bash
  git tag v1.0.0
  git push origin v1.0.0
  ```
  GitHub จะ build `.exe` แล้ว**แนบไฟล์ใน Releases อัตโนมัติ** — คนอื่นโหลดไปใช้ได้โดยไม่ต้องลง Python
- หรือกดรันเองที่แท็บ **Actions → Build Windows EXE → Run workflow** แล้วโหลดไฟล์จาก **Artifacts**
